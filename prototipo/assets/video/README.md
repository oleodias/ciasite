# Vídeo do Hero (entrada do site)

Coloque o vídeo institucional AQUI, com este nome exato:

    ciamed-hero.mp4

(Opcional, melhora compatibilidade/peso: ciamed-hero.webm)

## Como o código usa
O hero usa <video autoplay muted loop playsinline poster="admin-ajax.jpg">.
- ANTES de subir o vídeo: aparece a foto "Respire" (admin-ajax.jpg) como poster — o hero já fica completo.
- DEPOIS de subir ciamed-hero.mp4 aqui: o vídeo passa a tocar automaticamente, sem mexer no código.

## Specs recomendadas (para web)
- Resolução: 1920x1080 (ou 1280x720 se precisar aliviar)
- Formato: MP4 (codec H.264) + opcional WEBM
- Duração: loop curto, ~10 a 30s
- Sem áudio (o hero é mudo/autoplay)
- Peso: idealmente < 10–15 MB (posso orientar a compactar se estiver grande)

Como o arquivo é grande, você pode subir direto por aqui depois — o código já está preparado.
